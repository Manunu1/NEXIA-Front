# ==========================================================
# PROYECTO
# ==========================================================

Nombre:
NEXIA

Descripción:
Este proyecto consiste en el desarrollo de una plataforma educativa virtual bajo un modelo SaaS (Software as a Service), creada como proyecto final por cinco estudiantes de 5.º año de la especialidad Informática de ORT Argentina.
La plataforma está orientada a instituciones educativas que actualmente no cuentan con un campus virtual o sistema integral de gestión académica. Cada institución dispone de un espacio completamente independiente y personalizable, con su propia identidad visual, usuarios, materias y configuración, compartiendo una única infraestructura tecnológica.
El sistema busca centralizar toda la gestión educativa en una única plataforma, facilitando la comunicación entre alumnos, docentes y directivos, la organización de contenidos, la administración de tareas y evaluaciones, y el seguimiento del progreso académico.
Entre sus funcionalidades principales se incluyen:
Dashboard personalizado para cada tipo de usuario.
Calendario académico.
Mensajería entre docentes y alumnos.
Chats grupales por materia.
Comunicados institucionales.
Gestión de materias.
Publicación de contenidos educativos (PDF, presentaciones, enlaces, videos, etc.).
Entrega y corrección de trabajos prácticos.
Formularios, actividades y evaluaciones.
Sistema de notificaciones.
Administración de usuarios, cursos y permisos.
El principal diferencial del proyecto es la incorporación de una inteligencia artificial con enfoque pedagógico. A diferencia de los asistentes tradicionales, esta IA no proporciona respuestas directas a las consignas de los alumnos, sino que ofrece el contexto, las explicaciones y la información necesaria para que puedan desarrollar su propio razonamiento. Además, tiene acceso al material publicado por los docentes, permitiendo brindar una ayuda contextualizada según cada materia.
El proyecto se desarrolla con un fuerte enfoque en arquitectura de software, escalabilidad, seguridad, mantenibilidad, accesibilidad y experiencia de usuario, buscando construir una aplicación de calidad profesional preparada para crecer y ser utilizada por múltiples instituciones educativas.


Objetivo principal:
Desarrollar una plataforma educativa moderna, escalable y de alta calidad que permita digitalizar la gestión académica de instituciones educativas, mejorando la comunicación, la organización y el seguimiento del aprendizaje de los estudiantes.
El proyecto busca:
Reducir la brecha tecnológica entre instituciones públicas y privadas.
Centralizar la gestión académica en una única plataforma.
Mejorar el seguimiento del rendimiento y las actividades de los alumnos.
Facilitar el trabajo diario de docentes y directivos.
Fomentar el aprendizaje significativo mediante una inteligencia artificial con enfoque pedagógico.
Diseñar una arquitectura robusta, escalable y mantenible que permita incorporar nuevas funcionalidades en el futuro sin comprometer la calidad del sistema.
Todas las decisiones técnicas, de diseño y de arquitectura deben priorizar la calidad del software, la experiencia del usuario, la seguridad, la accesibilidad y la mantenibilidad por encima de soluciones rápidas o temporales.

# ==========================================================
# COLOR
# ==========================================================

## Colores primarios

- Azul marino (#1A237E)
- Color principal de la marca.
- Se utiliza para navegación, títulos, elementos importantes e identidad visual.

- Naranja (#FF9800)
- Color de acento.
- Se utiliza para botones principales, llamados a la acción, estados activos y elementos destacados.

## Colores secundarios

- Azul oscuro (#0D1654)
- Azul medio (#283593)
- Azul claro (#3949AB)

Se utilizan para gradientes, variaciones de componentes, hover states y jerarquía visual.

## Colores neutros

- Blanco (#FFFFFF)
- Gris muy claro (#F8F9FC)
- Gris azulado (#F2F3FB)
- Bordes (#E2E5F0)

Se utilizan para fondos, tarjetas, separadores y superficies.

## Colores de texto

- Principal: #071120
- Secundario: #1E2C4A
- Terciario: #5B6C8A
- Texto deshabilitado: #98A3BC

## Reglas

- El azul debe representar confianza, educación y profesionalismo.
- El naranja debe utilizarse únicamente como color de énfasis.
- Mantener abundante espacio en blanco.
- Evitar interfaces oscuras por defecto.
- No abusar del color naranja; debe llamar la atención únicamente cuando sea necesario.
- Utilizar gradientes suaves únicamente entre los tonos de azul y, ocasionalmente, entre los tonos de naranja.
- Los fondos deben permanecer claros para maximizar la legibilidad y la sensación de limpieza. 

Nunca usar colores saturados sin motivo.
